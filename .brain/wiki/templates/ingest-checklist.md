# Ingest Checklist

Use this checklist for every source ingest.

- [ ] Capture or reference the source under `raw/`.
- [ ] Create or update one page under `wiki/sources/`.
- [ ] Extract concepts, entities, claims, and questions.
- [ ] Search for existing pages before creating new ones.
- [ ] Update relevant concept/entity/claim/synthesis pages.
- [ ] Add useful backlinks.
- [ ] Update `wiki/index.md`.
- [ ] Append `wiki/log.md`.
- [ ] Report changed files and open questions to the user.
