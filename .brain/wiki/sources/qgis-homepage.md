# QGIS Homepage

## Summary

The QGIS homepage presents QGIS as a free and open-source GIS platform for spatial visualization, decision-making, map creation, layer editing, processing/analysis, and map sharing. For the user's workflow, the strongest signals are QGIS's cartographic design features, layout designer for high-quality large-format print maps, atlas/report support, analysis toolbox, format support, and standards/interoperability.

## Key Points

- QGIS emphasizes creating maps for desktop, mobile, cloud, and print media.
- It highlights cartographic design options and professional map production through an intuitive layout designer.
- It supports atlases and reports combining maps and tabular content.
- It includes editing/digitizing tools for points, lines, polygons, and meshes.
- It includes analysis workflows through an integrated toolbox, visual workflow composition, and third-party tools.
- It emphasizes data format support, standards/interoperability, web services, and publishing beyond the desktop.
- The captured homepage announces QGIS 4.0; feature decisions should still be checked against the current user manual before operational use.
