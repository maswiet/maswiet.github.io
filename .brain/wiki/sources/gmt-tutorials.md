# GMT Tutorials

## Summary

GMT Tutorials is a structured learning resource for GMT and PyGMT map production. For the user's map workflow, its main value is not one isolated command example, but a broader syllabus covering first maps, topography coloring, map layout, hillshade, map elements, CPT/colorbar editing, 3D maps, focal mechanisms, image draping, raster processing, and PyGMT notebook examples.

## Key Points

- The source contrasts GMT's command-line style with common GIS software such as QGIS and ArcGIS, which helps frame the GMT-vs-QGIS workflow question.
- It positions GMT as worth learning for users with many plotting tasks or a need to script map production.
- It states that chapters aim to build high-quality maps from scratch and teach organization through command-line, shell-script, or PyGMT/Python workflows.
- The table of contents includes map topics directly aligned with the user's needs: color the topography, design map layout, hillshade map, add map elements, edit CPT and colorbar, and drape an image over topography.
- The site indicates code and notebooks are MIT licensed and content is CC BY 4.0, which may support adapting examples into local templates with attribution.
