# QGIS

## Summary

QGIS is a free and open-source GIS platform. In the user's geophysical map workflow, QGIS is relevant for GIS data management, layer editing, visual inspection, styling, professional layout design, atlas/report generation, data-format interoperability, and publishing workflows.

## Relevance To User Workflow

- Strong candidate for preparing and checking spatial layers before final figure production.
- Strong candidate for cartographic layout, large-format print maps, report maps, atlases, and maps that require many GIS data sources.
- Useful when visual workflow composition and third-party processing tools are more ergonomic than a pure scripting workflow.
- Less directly evidenced here for geophysics-native scripted figure reproduction than GMT/PyGMT; specific QGIS documentation should be ingested before building operational templates.
