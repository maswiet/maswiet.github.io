# User Requires Publication-Quality Geophysical Maps

## Claim

The user expects future geophysical map outputs to meet a high cartographic standard and to include shaded topography/bathymetry, a legend, a north indicator, and an index map when appropriate.

## Evidence

- The user stated that they will often draw analysis results as maps and have a high standard for map quality.
- The user explicitly named topographic and bathymetric shading, legend, north arrow, and index map as needed elements.
- The user framed GMT as the established geophysics standard while noting growing QGIS use.
