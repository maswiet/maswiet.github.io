# GMT Tutorials Provide Map Production Learning Path

## Claim

GMT Tutorials provides a structured learning path for high-quality map production that is relevant to the user's need for recurring geophysical maps with topography/bathymetry shading, layout polish, map elements, and reproducible scripts.

## Evidence

- The landing page lists chapters on first maps, topography coloring, raster visualization, vector data, map layout, hillshade, contours/profiles, map elements, CPT/colorbar editing, 3D maps, focal mechanisms, and image draping.
- The source explicitly frames GMT as valuable for many plotting tasks and script-based map production.
- The source describes chapter goals as producing high-quality maps from scratch using GMT command-line, shell scripts, or PyGMT/Python scripts.
