# PyGMT Connects Python Analysis To GMT-Quality Maps

## Claim

PyGMT can serve as a bridge from Python-based geophysical analysis to GMT's high-quality static map and figure output.

## Evidence

- The PyGMT overview describes PyGMT as a Python wrapper for GMT.
- It attributes spatial data processing and high-quality plots/maps to the GMT/PyGMT toolchain.
- It emphasizes static vector graphics for publications, posters, talks, and export formats such as PDF and PNG.

## Confidence

Needs review for operational details because the captured overview is version `v0.1.0`. The conceptual claim is useful, but API examples and install instructions should come from current PyGMT documentation before template work.
