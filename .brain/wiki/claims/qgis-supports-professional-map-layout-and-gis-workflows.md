# QGIS Supports Professional Map Layout And GIS Workflows

## Claim

QGIS is relevant to high-quality map production because it supports cartographic design, professional layout, atlases/reports, layer editing, analysis workflows, format interoperability, and map publishing.

## Evidence

- The QGIS homepage highlights cartographic design features for desktop, mobile, cloud, and print media.
- It identifies professional map production through a layout designer and support for high-quality large-format print maps.
- It advertises atlas/report creation, layer editing, integrated analysis workflows, third-party tools, broad format support, standards/interoperability, and publishing workflows.
