# Elegant minimalist

Elegant minimalist is the Design System for Wiwit Suryanto's personal academic website. It supports personal branding, research portfolio material, and lecture-related content for the Geophysics UGM context.

The system is restrained, editorial, and academic: warm paper neutrals, deep charcoal text, a sparse volcanic red accent, Source Serif 4 for scholarly voice, and Inter for interface clarity.

## Files

| File | Description |
|------|-------------|
| `DESIGN.md` | Design System guidance in DESIGN.md format, with structured token frontmatter and human-readable sections. |
| `design_tokens.json` | Design Tokens Community Group-style token JSON derived from the same intended token set. |
| `tailwind.config.js` | Tailwind CSS v3 theme extension scaffold for colors, typography, radius, and spacing tokens. |
